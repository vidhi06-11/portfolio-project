# Portfolio Setup Project 
## Tools Installed 
- Cursor IDE 
- Claude Code Extension 
- GitHub
## Steps Completed 
1. Download and installed Cursor IDE.
2. Installed Claude Code extensions.
3. Installed Codex extension.
4. Created a public Github repository.
5. Connected the repository with Cursor.
6. Created and updated the README.md file.
## Issues Faced and Solutions 
1. Windows Defender showed a security warning while installing Cursor.
- Solution: Clicked "More info" and proceeded with the installation.
2. Git Clone command was not working.
- Solution: Installed Git and restarted Cursor.
## Learning Outcome
This project helped me learn how to use Cursor IDE, GitHub repositories, Git version control, and AI coding assistants.
